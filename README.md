# ReadyTogether

ReadyTogether is a privacy-first emergency planner that turns a household's real needs into a clear, downloadable action plan in under a minute.

## Why it exists

Emergency guidance is usually generic. A household managing medication, a powered medical device, mobility support, a young child, or a service animal needs more specific preparation. ReadyTogether creates that personalized plan without requiring an account or sending personal answers to a server.

## Features

- Plans for severe storms, wildfires, power outages, and floods
- Personalized actions for six household support needs
- On-device saving with browser local storage
- Downloadable and print-friendly plans
- Responsive, keyboard-friendly interface
- Reduced-motion and accessible focus support

## Run locally

```bash
npm install
npx next dev
```

Then open `http://localhost:3000`.

## Production build

```bash
npx next build
```

## Technology

React, TypeScript, Next.js, HTML, CSS, browser local storage, and Vercel.
